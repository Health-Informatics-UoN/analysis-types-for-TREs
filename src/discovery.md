# Discovery

When using Five Safes TES, the user does not have access to view the data (eyes off) prior to analysis, which makes it considerably more difficult to analyse the data. How does a researcher know what analysis to perform, or even if the data is appropriate for their work? The answer is to use summary data of the data set, a process known as cohort discovery.
